## 📋 What is Versiona?

Versiona helps you keep track of changes in your important data. It works like a control system for your software’s state and context, making it easy to manage different versions of information. Whether you use AI agents, enterprise databases, or frontend apps, Versiona stores and checks data changes cleanly and safely.

You don’t need any programming skills to use it. Versiona manages versions automatically, so your work stays organized, and you can review or return to previous states when needed.

## 🗂 How to Use Versiona

### Track Your Data Versions

- To start, open Versiona and select where your data lives. This could be a database or files.
- The app will scan your data and create the first version.
- Each time your data changes, Versiona adds a new version automatically.
- You can see all previous versions by clicking the “History” tab.
- If you want to restore any previous data state, choose the version and select “Restore.”

### Audit Your Changes

- Versiona keeps a log of every change made.
- You can review who made changes and when.
- Use this to find errors or unwanted updates in your data.
- This audit trail helps with compliance and reporting.

### Work with AI Agents

- Versiona supports storing multiple AI agent states.
- You can switch between agents or save different scenarios.
- This helps test and compare AI behaviors over time.

## ⚙️ Configuring Versiona

You can change a few settings to fit your work style:

- **Database Connection:** Set up your PostgreSQL or other database details to save versions there.
- **Storage Limits:** Control how much disk space versions can use.
- **Notifications:** Enable alerts when big changes happen.
- **Backup Options:** Schedule backups for your version data.

Access these options from the Settings menu inside Versiona.

## 🛠 Troubleshooting

- **Installer won’t run:** Check if Windows SmartScreen or antivirus blocks the file. Allow it manually.
- **App won’t start:** Restart your computer, then try again.
- **Versions missing:** Make sure your data source is connected and accessible.
- **Slow performance:** Close other programs to free memory, or increase storage limits in settings.
